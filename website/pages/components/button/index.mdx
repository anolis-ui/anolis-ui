# Button

```tsx
import { Button } from "anolis-ui";

interface ButtonProps extends ComplementProps, SystemProps {

}

type ButtonVariant = "solid" | "clear" | "outline" | "link";
type ButtonSize = "xs" | "sm" | "md" | "lg";
```

Button has 4 default variants and sizes.

## Variants

You can use 4 variants of button. Just use **v** prop.

<ComponentDisplay>
  {[
    [<Button>Solid</Button>, "<Button>Solid</Button>"],
    [<Button v="outline">Outline</Button>, "<Button v=\"outline\">Outline</Button>"],
    [<Button v="clear">Clear</Button>, "<Button v=\"clear\">Clear</Button>"],
    [<Button v="link">Link</Button>, "<Button v=\"link\">Link</Button>"]
  ]}
</ComponentDisplay>

## Sizes

You can use 4 sizes of button. Just use **s** prop.

  {[
