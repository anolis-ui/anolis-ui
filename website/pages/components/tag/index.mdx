# Tag

```tsx
import { Tag } from "anolis-ui";

export type TagVariant = "solid" | "outline" | "clear";

export interface TagThemeProps extends ComplementThemeProps, PseudoProp {

}
```
Tag component for labeling, categorizing etc.

<ComponentDisplay>
  {[
    [<Tag>Solid tag</Tag>, "<Tag>Solid tag</Tag>"],
  ]}
</ComponentDisplay>


## Variants

You can use 4 variants of tag. Just use **v** prop.

<ComponentDisplay>
  {[
    [<x.div spaceX="4">
      <Tag>Solid tag</Tag>
      <Tag v="outline">Outline tag</Tag>
      <Tag v="clear">Clear tag</Tag>
    </x.div>,
    `
<x.div spaceX="4">
  <Tag>Solid tag</Tag>
  <Tag v="outline">Outline tag</Tag>
  <Tag v="clear">Clear tag</Tag>
</x.div>
    `],
  ]}
</ComponentDisplay>


## With icon

Icon can be added to the Tag, both left and right positions

<ComponentDisplay>
  {[
    [<Tag leftIcon={<DownIcon/>} _leftIcon={{ fill: "#fff" }} rightIcon={<DownIcon/>} _rightIcon={{ fill: "#fff" }}>Solid tag</Tag>,
    `
<Tag leftIcon={<DownIcon/>} rightIcon={<DownIcon/>}>Solid tag</Tag>
    `
    ],
  ]}
</ComponentDisplay>
