# Typography

```tsx
import { Typography } from "anolis-ui";

export type TypographyVariant = never;
export type TypographySize = never;

export interface TypographyThemeProps extends PseudoProp {
  _h1?: PseudoProp;
  _h2?: PseudoProp;
  _h3?: PseudoProp;
  _h4?: PseudoProp;
  _h5?: PseudoProp;
  _h6?: PseudoProp;
  _p?: PseudoProp;
  _lead?: PseudoProp;
}
```

## Basic

Because all app is wrapped by AnolisProvider, you can use just x.components.

<ComponentDisplay>
  {[
    [
      <x.p>Paragraph</x.p>,
      `
<x.p>Paragraph</x.p>
      `
    ]
  ]}
</ComponentDisplay>

## Txt component

You can also use Txt component, which has **t prop** to set type of Typography.
Also Lead variant can be set by using **Txt** component.

<ComponentDisplay>
  {[
    [
      <Txt t="lead">Lead/perex paragraph</Txt>,
      `
<Txt t="lead">Lead/perex paragraph</Txt>
      `
    ]
  ]}
</ComponentDisplay>
